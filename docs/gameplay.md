# Gameplay

## Reaction Test

The player waits for a target stimulus.

Goals:
- measure reaction timing
- detect impulsive false presses
- evaluate consistency

Current implementation:
- Adjustable number of rounds
- randomized delay
- moving visual target

---

## Memory Test

The player repeats a number sequence.

Goals:
- evaluate short-term memory
- detect sequencing errors

Current implementation:
- fixed sequence
- keyboard number input
